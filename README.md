# Password Manager

A simple GUI-based Password Manager built with Python and Tkinter.  
It allows you to:

- Generate strong random passwords.
- Save website credentials (website, email, password) to a local file.
- Customize the number of letters, symbols, and numbers in the password.

## Features

- Generate passwords with a mix of letters, numbers, and symbols.
- Save credentials in a `data.txt` file.
- Easy-to-use interface using Tkinter.
- Logo display using PIL (Pillow).

## Requirements

- Python 3.x
- Tkinter (usually comes pre-installed with Python)
- Pillow library for image handling:

pip install pillow
